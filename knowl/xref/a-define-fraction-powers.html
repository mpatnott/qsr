Activity 14.2.1.

We already know that \(x^2 = x\cdot x\text{,}\) \(x^3 = x\cdot x \cdot x\text{,}\) etc. We also know that
🔗
\begin{equation*} \sqrt{x}\sqrt{x} = x \end{equation*}
\begin{equation*} \sqrt[3]{x}\sqrt[3]{x}\sqrt[3]{x} = x \end{equation*}
\begin{equation*} \sqrt[4]{x}\sqrt[4]{x}\sqrt[4]{x}\sqrt[4]{x} = x \end{equation*}
and so on.
🔗
Thinking about exponents as repeated multiplication only works when the power is a whole number! How can we think about what it means to take the \(\frac13\)rd power, like in Prep Activity 14.1.1? The exponent properties can help us!
🔗
